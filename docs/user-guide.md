## 1. 这是什么

Tianxuan 是一个运行在手机上的 [1Panel](https://1panel.cn) 管理工具。

1Panel 是 Linux 服务器面板（类似宝塔），它的网页端已经很漂亮了，但有时打开电脑不方便——手机想看一眼服务器状态、重启个网站、浏览一下文件，这时候 Tianxuan 就派上用场了。

**支持的平台：**
- Android (APK)
- Web (浏览器打开)

**前提：** 你的服务器上已经安装了 1Panel（v2.x），并开启了 API 访问。

## 2. 安装

### Android

1. 从 [GitHub Releases](https://github.com/CHINAYYDSNB/Tianxuan/releases/latest) 下载最新的 `Tianxuan-*.apk`
2. 在手机文件管理器点开安装
3. 如果提示"未知来源"，去设置里允许安装未知应用

> APK 约 57MB，首次安装后不需要额外数据下载。

### Web

```bash
# 如果你的设备上安装了 Flutter 开发环境
git clone https://github.com/CHINAYYDSNB/Tianxuan.git
cd Tianxuan
flutter run -d chrome --web-port 25568
```

> 注意：Web 版需要配合 `server.mjs` 同源代理来绕过 CORS 限制。

## 3. 连接你的 1Panel 服务器

### 第一步：在 1Panel 后台获取 API Key

1. 登录你的 1Panel 网页管理端
2. 进入「设置」→「API Key」
3. 点击「生成 Key」，复制生成的密钥

### 第二步：在 Tianxuan 中输入连接信息

1. 打开 Tianxuan App
2. 填写以下信息：

| 字段 | 说明 | 示例 |
|------|------|------|
| **服务器地址** | 你的 1Panel 访问地址 | `http://192.168.1.100:25567` |
| **API Key** | 上一步复制的密钥 | `xxxxxxxxxxxxxxxx` |
| **HTTPS** | 如果有配置 SSL 就勾上 | 默认不勾 |

3. 点击「连接测试」→ 成功后会跳转到主界面

### 多服务器

在「设置」页可以添加多台服务器，随时切换。

> **注意：** API Key 相当于服务器管理员密码，不要传给不信任的人。Tianxuan 会将 Key 加密存储在本地。

## 4. 功能介绍

底部导航栏一共 6 个 Tab：

### 📊 概览

![概览界面]（待补充截图）

显示服务器的基本信息：
- **CPU 使用率** — 环状图 + 百分比
- **内存使用率** — 总量 / 已用 / 百分比
- **磁盘使用率** — 总量 / 已用 / 百分比
- **运行时间** — 服务器连续运行时长（实时跳动）
- **系统信息** — Hostname、操作系统、内核版本

下拉刷新获取最新数据。

### 📁 文件

文件浏览器，类似于手机上的文件管理器：
- **浏览** — 进入目录，查看文件列表
- **创建** — 新建文件夹或文件
- **编辑** — 在线编辑文本文件（代码、配置等）
- **重命名/移动/复制** — 文件操作
- **删除** — 单文件或批量删除
- **压缩解压** — zip 格式打包和解压
- **上传/下载** — 从手机上传输文件到服务器

### 🌐 网站

管理 1Panel 中的网站：
- 查看网站列表（域名、状态、类型）
- 启动/停止/重启网站
- 删除网站（滑动删除）

状态标签一目了然：绿色=运行中，红色=已停止。

### 🐳 容器

Docker 管理功能：

| 子页 | 功能 |
|------|------|
| **容器** | 容器列表 → 查看详情(资源占用) → 启停操作 → 实时日志(SSE) |
| **镜像** | 镜像列表 → 拉取新镜像 → 删除镜像 |
| **Compose** | Compose 项目列表 → 启停管理 |
| **应用商店** | 浏览 1Panel 应用商店 → 查看详情和版本 → 安装应用 |
| **已安装** | 查看已安装应用 → 详情(环境变量) → 启停 → 可更新标记 |

### 🛡️ WAF

Web 应用防火墙管理（1Panel 专业版功能）：
- **概览** — WAF 运行状态
- **IP 规则** — IP 黑白名单
- **规则组** — WAF 规则配置
- **日志** — 拦截记录

### ⚙️ 设置

- **服务器管理** — 切换/添加/删除服务器
- **健康检测** — 查看系统健康状态，配置告警阈值
- **关于** — 版本信息、GitHub 链接、更新检测

### 附加功能

- **SSH 终端** — 在 App 内打开远程 Shell 连接
- **Logto 登录** — 可选身份认证（OIDC 协议，需自行部署 Logto 服务）
- **更新检测** — 自动检查 GitHub 新版本

## 5. 常见问题

### Q: 连接测试提示"连接失败"

- 检查 IP 和端口是否正确
- 确认 1Panel 服务正在运行
- 如果服务器有防火墙，检查是否放行了对应端口
- 如果使用 HTTPS，确认 SSL 证书配置正确

### Q: 登录后页面空白

- 下拉刷新试试
- 检查服务器是否正常运行
- 如果持续空白，在设置中重新连接

### Q: Web 版 CORS 报错

Web 浏览器有跨域安全限制。使用项目根目录的 `server.mjs`：

```bash
node server.mjs
```

然后在浏览器打开 `http://localhost:25568`。

APK 无此问题。

### Q: 数据安全吗？

- API Key 用 `flutter_secure_storage` 加密存储（Android Keystore）
- 所有通信走 HTTP/HTTPS，加密取决于你的 1Panel 配置
- 建议：不要在公共 WiFi 下使用 HTTP 连接

### Q: 能管理多个 1Panel 服务器吗？

可以。在「设置」页添加多台服务器，保存后可随时切换。

### Q: 支持 1Panel 全部功能吗？

目前覆盖了最常用的管理功能（服务器监控、网站、文件、Docker、WAF）。部分高级功能（数据库、计划任务、日志审计等）尚未集成，会持续更新。
